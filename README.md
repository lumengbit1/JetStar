# JetStar Program

## Getting started

#### Requirement

This app uses node v15.4.0, or npm v15.4.0

#### Project setup

Install project dependencies you can use npm:

```
npm install
```

#### Run development server

```
npm start
```

#### Run linters

```
npm run lint
```

#### Run tests

```
npm test
```

#### Run Cypress

```
npm run cypress:open
```

#### Create production build

```
npm run build
```
